Welcome to Automacorp's frontend documentation!
===================================

Automacorp is the final project of a programming course. It is an app serving to manage buildings, rooms, windows, heaters and temperature sensors.
It pulls data from the `Automacorp backend <https://automacorp-erwin-martin.cleverapps.io/>` deployed on Clevercloud.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
